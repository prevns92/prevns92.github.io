# 合成オッズ計算機

「合成オッズ計算機」は、複数の買い目のオッズから合成オッズを素早く計算し、資金配分をサポートするiOSアプリケーションです。

## 主な機能
- 複数オッズの入力による合成オッズの即時計算
- 各買い目への資金配分シミュレーション
- 過去の計算結果保存
